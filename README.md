# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**.

## ✨ Features

* Two-player gameplay (X and O)
* Displays the winner (for example, **"Winner is X"**)
* Detects and announces a draw
* Reset Game button to restart the game at any time
* New Game button that appears after the game ends and starts a fresh game

## 🛠️ Technologies Used

* **HTML** – Provides the structure of the game.
* **CSS** – Styles the game and creates a clean, user-friendly interface.
* **JavaScript** – Implements the game logic, handles user interactions through event listeners, detects winners and draws, and manages the game state.

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your web browser to play the game.
